# BigInteger and Rational

Classes for working with very large numbers and fractions.

**Key features:**
* `BigInteger` can store huge numbers and do math operations (`+`, `-`, `*`, `/`, `%`).
* `Rational` works with fractions using `BigInteger`.
* Fractions are automatically shortened using GCD (Greatest Common Divisor).
* Can convert fractions to decimal strings with a given precision.
